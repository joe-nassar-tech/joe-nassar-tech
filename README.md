## Professional Introduction

I am a Junior Full-Stack Developer from Lebanon with 3 years of hands-on learning and project-building experience. I build practical web applications with Laravel, React, TypeScript, REST APIs, dashboards, authentication, real-time features, and clean user interfaces.

My work focuses on business-oriented products: e-commerce platforms, auction systems, POS tools, admin panels, authentication flows, and portfolio systems. I use structured prompting to move faster, then test, refine, and take responsibility for the final behavior.

## About Me

- Based in Lebanon and open to remote, hybrid, and local junior full-stack opportunities.
- Strongest stack: Laravel, PHP, React, TypeScript, JavaScript, Tailwind CSS, MySQL, Redis.
- I enjoy turning product ideas into usable systems with clear admin workflows and practical business logic.

- I am comfortable working with AI tools through structured prompting, code review, testing, and refactoring.
